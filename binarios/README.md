# Binarios

Esta carpeta esta reservada para los artefactos generados del firmware. Los archivos presentes pueden provenir de compilaciones locales previas.

- `boot_app0.bin`
- `bootloader.bin`
- `firmware.bin`
- `partitions.bin`
- `rf-kill-esp32-devkit-web-v2.bin`

El archivo unificado `rf-kill-esp32-devkit-web-v2.bin` es el que referencia `manifest.json` para el Web Flasher.
