ESP32-C3 SuperMini

RF-KILL Web Flasher

Instalador web para cargar el firmware del proyecto RF-KILL en un ESP32-C3 SuperMini. El firmware inicia automaticamente al energizar la placa y muestra el estado de diagnostico por monitor serie.

Conecta el ESP32-C3 por USB, abre esta pagina en Chrome o Edge de escritorio y presiona instalar.

Pasos recomendados

Usa un cable USB de datos, no solo de carga.
Selecciona el puerto serial del ESP32-C3 cuando el navegador lo solicite.
Despues del flasheo, abre el monitor serie a 115200 baudios para revisar el diagnostico.
