Web Flasher

Carga el firmware del proyecto directamente desde el navegador. Conecta la placa por USB, selecciona el puerto serial y deja que el instalador escriba los binarios preparados para el ESP32-WROOM DevKit.

STATUS: READY // BAUD: 115200 // TARGET: ESP32
01_BOOTOK
  • Usa Chrome o Edge de escritorio.
  • Conecta un cable USB de datos.
  • Selecciona el puerto del ESP32.
02_FLASHBIN
  • bootloader.bin en 0x0000.
  • partitions.bin en 0x8000.
  • firmware.bin en 0x10000.
03_SERIAL115200

Despues del flasheo abre el monitor serie para revisar el arranque headless y el diagnostico de los modulos nRF24L01+.

DEVICE_MAP2x NRF24L01+

El montaje usa dos radios nRF24L01+ sobre un bus SPI compartido. Cada modulo conserva sus pines CE y CSN independientes para control desde el ESP32-WROOM DevKit.

SCK=GPIO18 // MISO=GPIO19 // MOSI=GPIO23 // CE1=GPIO5 // CSN1=GPIO17 // CE2=GPIO16 // CSN2=GPIO4
